# AgentOffice 项目文档

## 一、项目概述

AI数字员工云端开发与运维平台，采用前后端分离架构。

### 技术栈

**后端**
- Spring Boot 3.2.0
- Java 17
- MyBatis 3.0.3
- MySQL
- JWT (jjwt 0.12.3)

**前端**
- React 19
- Ant Design 6
- Tailwind CSS 3.4
- ECharts 6
- Zustand 5
- React Router 7
- Vite 8

## 二、项目结构

```
AgentOffice/
```

## 三、数据库

### 表结构

1. **sys_user** - 用户表
2. **agent_employee** - 员工表
3. **office_desk** - 工位表
4. **task_info** - 任务表
5. **task_step** - 任务步骤表
6. **dev_project** - 项目表
7. **dev_file** - 文件表
8. **deploy_service** - 服务表
9. **operation_log** - 操作日志表
10. **chat_session** - 会话表
11. **chat_message** - 消息表

### 初始化

```bash
mysql -u root -p < docs/sql/init.sql
```

## 四、API接口

### 认证模块 `/api/auth`
- POST /login - 用户登录
- POST /register - 用户注册
- GET /current - 获取当前用户
- POST /logout - 退出登录

### 员工管理 `/api/employees`
- GET / - 员工列表
- GET /:id - 获取员工详情
- POST / - 创建员工
- PUT /:id - 更新员工
- DELETE /:id - 删除员工
- PATCH /:id/status - 更新员工状态

### 任务管理 `/api/tasks`
- GET / - 任务列表
- GET /:id - 任务详情
- POST / - 创建任务
- PUT /:id - 更新任务
- DELETE /:id - 删除任务
- PATCH /:id/progress - 更新进度
- PATCH /:id/status - 更新状态
- POST /:id/assign - 分配任务

### 虚拟办公室 `/api/office`
- GET /layout - 获取办公室布局
- GET /employees/status - 员工状态概览

### 云端开发 `/api/dev`
- GET /projects - 项目列表
- GET /projects/:id - 项目详情
- POST /projects - 创建项目
- PUT /projects/:id - 更新项目
- DELETE /projects/:id - 删除项目
- GET /projects/:id/files - 文件树
- GET /files/:id - 文件内容
- PUT /files/:id - 更新文件
- POST /run - 运行代码

### 部署运维 `/api/deploy`
- GET /services - 服务列表
- GET /services/:id - 服务详情
- POST /services - 创建服务
- PUT /services/:id - 更新服务
- DELETE /services/:id - 删除服务
- POST /services/:id/start - 启动服务
- POST /services/:id/stop - 停止服务
- POST /services/:id/restart - 重启服务
- GET /services/:id/logs - 服务日志

### 数据分析 `/api/analytics`
- GET /dashboard - Dashboard数据
- GET /employees - 员工效率
- GET /tasks/trend - 任务趋势
- GET /kpi - KPI数据

## 五、启动方式

### 后端启动

```bash
# 确保MySQL运行并已导入数据
cd AgentOffice
mvn spring-boot:run
```

### 前端启动

```bash
cd frontend
npm install
npm run dev
```

### 默认账号

- 用户名: admin
- 密码: admin123

## 六、界面功能

### 登录页
- 左右布局：左侧品牌展示，右侧登录表单
- 默认填充admin/admin123便于测试

### 主控制台 (Dashboard)
- KPI卡片：任务完成率、总任务数、完成任务数、平均效率
- 任务趋势图
- 员工状态分布
- 最近任务列表

### 虚拟办公室
- 2.5D风格的工位网格
- 每个工位显示员工头像、姓名、角色、状态
- 点击空闲工位可创建新员工
- 点击有员工工位显示详情抽屉

### 员工管理
- 员工列表表格
- 支持搜索和筛选
- 创建/编辑/删除员工

### 任务管理
- Tabs切换：全部/进行中/已完成/已失败
- 任务表格：任务名、优先级、执行人、状态、进度
- 创建任务弹窗
- 任务详情抽屉：步骤流、日志时间线

### 云端开发
- 左侧文件树（VSCode风格）
- 中间代码编辑器（深色主题）
- 右侧运行日志
- 支持创建项目、运行代码

### 部署运维
- 服务列表：服务名、状态、镜像、版本、端口、运行时间
- 操作按钮：启动/停止/重启/查看日志
- 服务详情抽屉：资源使用监控

### 数据分析
- KPI卡片
- 任务完成率仪表盘
- 任务趋势折线图
- 员工效率柱状图

### 系统设置
- 基本设置
- 通知设置
- 安全设置
