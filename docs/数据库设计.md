# AgentOffice 数据库设计

## 一、数据库概述

- **数据库名**: agent_office
- **字符集**: utf8mb4
- **排序规则**: utf8mb4_unicode_ci

## 二、表结构设计

### 1. 用户表 (sys_user)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| username | VARCHAR(50) | 用户名 |
| password | VARCHAR(255) | 密码(加密) |
| nickname | VARCHAR(50) | 昵称 |
| avatar | VARCHAR(255) | 头像URL |
| email | VARCHAR(100) | 邮箱 |
| phone | VARCHAR(20) | 手机号 |
| status | TINYINT | 状态(0禁用,1正常) |
| create_time | DATETIME | 创建时间 |
| update_time | DATETIME | 更新时间 |

### 2. 员工表 (agent_employee)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| name | VARCHAR(50) | 员工姓名 |
| avatar | VARCHAR(255) | 头像URL |
| role | VARCHAR(50) | 角色(开发/测试/运维/产品) |
| position | VARCHAR(50) | 职位 |
| status | VARCHAR(20) | 状态(工作中/思考中/编译中/部署中/完成/空闲/在线) |
| task_count | INT | 任务数 |
| efficiency | DECIMAL(5,2) | 效率百分比 |
| desk_id | BIGINT | 工位ID |
| create_time | DATETIME | 创建时间 |
| update_time | DATETIME | 更新时间 |

### 3. 工位表 (office_desk)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| desk_code | VARCHAR(20) | 工位编号 |
| row_num | INT | 排号 |
| col_num | INT | 列号 |
| employee_id | BIGINT | 员工ID(可为空) |
| status | TINYINT | 状态(0空闲,1占用) |
| create_time | DATETIME | 创建时间 |

### 4. 任务表 (task_info)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| task_name | VARCHAR(100) | 任务名称 |
| description | TEXT | 任务描述 |
| priority | VARCHAR(20) | 优先级(高/中/低) |
| executor_id | BIGINT | 执行人ID |
| status | VARCHAR(20) | 状态(进行中/已完成/已失败/待分配) |
| progress | INT | 进度(0-100) |
| create_user | BIGINT | 创建人 |
| create_time | DATETIME | 创建时间 |
| update_time | DATETIME | 更新时间 |
| end_time | DATETIME | 结束时间 |

### 5. 任务步骤表 (task_step)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| task_id | BIGINT | 任务ID |
| step_name | VARCHAR(100) | 步骤名称 |
| step_order | INT | 步骤顺序 |
| status | VARCHAR(20) | 状态(已完成/进行中/待处理) |
| complete_time | DATETIME | 完成时间 |

### 6. 项目表 (dev_project)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| project_name | VARCHAR(100) | 项目名称 |
| description | TEXT | 项目描述 |
| language | VARCHAR(20) | 语言类型 |
| owner_id | BIGINT | 负责人 |
| status | TINYINT | 状态 |
| create_time | DATETIME | 创建时间 |
| update_time | DATETIME | 更新时间 |

### 7. 文件表 (dev_file)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| project_id | BIGINT | 项目ID |
| file_name | VARCHAR(255) | 文件名 |
| file_path | VARCHAR(500) | 文件路径 |
| file_type | VARCHAR(20) | 文件类型 |
| content | LONGTEXT | 文件内容 |
| parent_id | BIGINT | 父目录ID |
| create_time | DATETIME | 创建时间 |
| update_time | DATETIME | 更新时间 |

### 8. 服务表 (deploy_service)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| service_name | VARCHAR(100) | 服务名称 |
| image | VARCHAR(255) | 镜像 |
| version | VARCHAR(50) | 版本 |
| status | VARCHAR(20) | 状态(运行中/已停止/异常) |
| port | INT | 端口 |
| container_id | VARCHAR(100) | 容器ID |
| cpu_usage | DECIMAL(5,2) | CPU使用率 |
| memory_usage | DECIMAL(5,2) | 内存使用 |
| running_time | BIGINT | 运行时间(秒) |
| create_time | DATETIME | 创建时间 |
| update_time | DATETIME | 更新时间 |

### 9. 操作日志表 (operation_log)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| user_id | BIGINT | 用户ID |
| action | VARCHAR(50) | 操作类型 |
| target_type | VARCHAR(50) | 目标类型 |
| target_id | BIGINT | 目标ID |
| detail | TEXT | 详情 |
| ip_address | VARCHAR(50) | IP地址 |
| create_time | DATETIME | 操作时间 |

### 10. 会话表 (chat_session)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| session_id | VARCHAR(100) | 会话ID |
| user_id | BIGINT | 用户ID |
| agent_id | BIGINT | Agent ID |
| create_time | DATETIME | 创建时间 |
| update_time | DATETIME | 更新时间 |

### 11. 消息表 (chat_message)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| session_id | BIGINT | 会话ID |
| role | VARCHAR(20) | 角色(user/assistant) |
| content | TEXT | 内容 |
| create_time | DATETIME | 创建时间 |

## 三、索引设计

1. sys_user: idx_username(username)
2. agent_employee: idx_status(status), idx_role(role)
3. task_info: idx_executor_id(executor_id), idx_status(status)
4. dev_file: idx_project_id(project_id)
5. deploy_service: idx_status(status)
6. operation_log: idx_user_id(user_id), idx_create_time(create_time)
