# AgentOffice

**AgentOffice** 是一个基于 **AI 数字员工（AI Agents）** 的云端开发与运维平台。
用户可以在虚拟办公室中创建并管理 AI 员工，让 AI 自动完成 **开发、测试、部署和运维等 DevOps 流程**。

系统通过多智能体协作的方式，让不同角色的 AI 员工共同完成软件开发任务，实现从需求到部署的自动化流程。

---

# 项目简介

传统软件开发需要多个角色协作，例如产品经理、开发工程师、测试工程师和运维工程师。本项目通过 **AI Agent 技术**模拟企业团队结构，使 AI 数字员工能够自动完成开发任务。

用户可以在虚拟办公室中：

* 创建 AI 员工
* 分配开发任务
* 监控任务执行过程
* 查看开发成果和系统运行状态

系统整体目标是构建一个 **AI 驱动的 DevOps 自动化平台**。

---

# 系统主要功能

## 1 用户管理模块

* 用户注册、登录及身份认证
* 用户信息管理与权限控制
* 每个用户对应独立的虚拟办公室空间

---

## 2 虚拟办公室展示模块

系统通过 **虚拟办公室界面** 展示 AI 员工的工作状态。

功能包括：

* 展示多个办公工位
* 每个工位对应一个 AI 数字员工
* 空工位可以创建新的员工
* 实时显示员工工作状态，例如：

```
工作中
思考中
编译中
部署中
已完成
```

---

## 3 AI 数字员工管理模块

用户可以创建不同角色的 AI 员工，例如：

* 开发工程师
* 测试工程师
* 运维工程师
* 产品经理

每个员工可以配置：

* 职责
* 权限
* 工作流程
* Prompt 角色设定

AI 员工具有独立记忆，可以记录任务历史和工作成果。

---

## 4 任务管理模块

用户可以创建任务并分配给 AI 员工，例如：

```
开发用户登录接口
编写数据库表结构
修复接口BUG
```

系统功能包括：

* 创建任务
* 自动分配任务给对应角色
* AI 执行任务
* 任务执行状态追踪

AI 可以执行：

* 代码生成
* 自动测试
* 自动部署

---

## 5 云端开发与运行模块

AI 员工可以在云端环境中完成代码开发和执行。

功能包括：

* AI 自动编写代码
* 自动编译和运行程序
* 自动执行测试
* 返回代码运行结果

---

## 6 自动化部署与运维模块

系统支持自动化部署与运维能力。

主要功能：

* 将代码构建为 Docker 镜像
* 自动部署到云端容器环境
* 服务运行监控
* 运维操作，例如：

```
重启服务
停止服务
查看日志
查看运行状态
```

---

## 7 成果展示与数据分析模块

系统会记录 AI 员工的工作成果，并进行统计分析。

功能包括：

* 查看员工完成的任务
* 展示代码、文档和测试报告
* 统计任务完成率
* 分析员工工作效率

数据以图表形式展示：

* 折线图
* 柱状图
* 饼图

# 系统架构

系统采用 **前后端分离 + AI Agent + DevOps 平台架构**。

```
用户浏览器
     │
React 前端系统
     │
REST API
     │
Spring Boot 后端服务
     │
├─ 用户管理
├─ 数字员工管理
├─ 任务调度系统
├─ DevOps 执行系统
│
数据库（MySQL / Redis）
     │
云端容器环境（Docker）
     │
大模型 API（LLM）
```

# 技术栈

## 后端技术

* Spring Boot：核心开发框架
* MyBatis：数据库持久层
* MySQL：存储任务、员工、日志等数据
* Redis：缓存任务状态与员工状态
* RESTful API：前后端通信
* LLM API：调用大模型完成任务

---

## 前端技术

* React：前端核心框架
* Ant Design：UI组件库
* React Router：路由管理
* Axios：HTTP请求工具
* ECharts：数据可视化
* Redux / Zustand：状态管理

# AI 智能体系统

系统中的每个数字员工都是一个 **Agent**。

Agent 的组成包括：

* 角色 Prompt
* 任务处理能力
* 工具调用能力
* 记忆系统
* Agent 之间的通信能力

AI 员工之间可以进行协作，例如：

```
产品经理 → 提出需求
开发工程师 → 编写代码
测试工程师 → 执行测试
运维工程师 → 部署系统
```

通过这种方式实现 **多智能体协作的 DevOps 自动化流程**。
