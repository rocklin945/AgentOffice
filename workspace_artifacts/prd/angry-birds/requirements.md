# 愤怒的小鸟 - 产品需求文档 (PRD)

## 1. 项目概述

| 字段 | 内容 |
|------|------|
| 项目名称 | 愤怒的小鸟 (Angry Birds) |
| 项目类型 | 弹射类物理小游戏 |
| 项目文件夹 | angry-birds |
| 核心功能 | 玩家操控弹弓发射小鸟，攻击猪猪堡垒获取胜利 |
| 目标用户 | 休闲游戏玩家 |

---

## 2. 游戏元素

### 2.1 角色

| 角色 | 描述 | 数量 |
|------|------|------|
| 小鸟 | 可发射的角色，不同颜色小鸟有不同技能 | 3种 |
| 猪猪 | 攻击目标，需要被消灭 | 若干 |

### 2.2 小鸟技能

| 小鸟颜色 | 技能描述 |
|----------|----------|
| 红色小鸟 | 普通飞行，无特殊技能 |
| 蓝色小鸟 | 飞行中点击可分裂成3只小分身 |
| 黄色小鸟 | 飞行中点击可加速冲刺 |

### 2.3 游戏场景

| 元素 | 描述 |
|------|------|
| 弹弓 | 发射小鸟的起始位置，左侧固定 |
| 地面 | 游戏底部边界 |
| 天空背景 | 蓝色渐变背景 |
| 砖块/木块/石头 | 可被破坏的建筑方块 |

---

## 3. 功能需求

### 3.1 核心玩法

- **弹弓发射**：鼠标/手指拖拽弹弓皮筋调整发射角度和力度，松开发射
- **物理引擎**：小鸟飞行遵循抛物线轨迹，受重力影响
- **碰撞检测**：小鸟撞击建筑物和猪猪产生物理反馈
- **关卡目标**：消灭所有猪猪即可通关

### 3.2 用户交互

| 交互 | 操作描述 |
|------|----------|
| 瞄准 | 鼠标按住小鸟并向后拖拽，显示瞄准线和力度 |
| 发射 | 松开鼠标，小鸟飞出发射 |
| 小鸟技能 | 在飞行过程中点击屏幕激活小鸟特殊技能 |
| 重新开始 | 点击重玩按钮重新开始当前关卡 |

### 3.3 游戏流程

```
开始界面 → 选择关卡 → 游戏加载 → 弹弓瞄准发射 → 小鸟飞行 → 碰撞检测判定 → 结算（通关/失败） → 下一关
```

### 3.4 界面UI

| 界面 | 元素 |
|------|------|
| 开始界面 | 游戏标题、开始按钮 |
| 游戏界面 | 弹弓、小鸟、关卡、分数、重玩按钮 |
| 结果界面 | 通关/失败提示、继续/重玩按钮 |

---

## 4. 技术需求

| 项目 | 描述 |
|------|------|
| 技术栈 | HTML5 Canvas + JavaScript |
| 游戏框架 | 原生实现（可选Phaser） |
| 响应式 | 支持桌面和移动端 |
| 音效 | 可选背景音乐和音效 |

---

## 5. 关卡设计

### 5.1 第1关

- **难度** | 简单
- **猪猪数量** | 2只
- **建筑** | 少量木块搭建的简易堡垒
- **通关条件** | 消灭2只猪猪

### 5.2 第2关

- **难度** | 中等
- **猪猪数量** | 3只
- **建筑** | 木块+砖块组合的堡垒
- **通关条件** | 消灭3只猪猪

### 5.3 第3关

- **难度** | 困难
- **猪猪数量** | 4只
- **建筑** | 木块+砖块+石头混合堡垒
- **通关条件** | 消灭4只猪猪

---

## 6. 计分系统

| 行为 | 分数 |
|------|------|
| 直接撞击猪猪 | 5000分 |
| 建筑方块倒塌压死猪猪 | 3000分 |
| 剩余小鸟 | 每只+10000分 |
| 关卡通关 | 总分累计 |

---

## 7. 验收标准

- [ ] 弹弓交互流畅，发射角度和力度准确
- [ ] 小鸟飞行轨迹符合物理抛物线
- [ ] 碰撞检测准确，建筑可被破坏
- [ ] 3种小鸟技能均可用
- [ ] 3个关卡均可游玩并通关
- [ ] 分数计算正确
- [ ] UI界面美观清晰
- [ ] 游戏无明显卡顿

---
