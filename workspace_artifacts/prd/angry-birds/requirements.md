# 愤怒的小鸟游戏 PRD

## 1. 项目概述

- **项目名称**：愤怒的小鸟（Angry Birds）
- **项目类型**：2D 物理解谜游戏
- **核心功能**：玩家使用弹弓发射小鸟，摧毁结构并消灭绿猪
- **目标用户**：休闲游戏玩家

## 2. 游戏功能需求

### 2.1 核心玩法

| 功能 | 描述 |
|------|------|
| 弹弓发射 | 拖拽弹弓释放小鸟，支持瞄准角度和力度控制 |
| 物理引擎 | 实现重力、碰撞、抛物线轨迹 |
| 小鸟类型 | 红鸟（基础）、黄鸟（加速）、蓝鸟（分裂） |
| 绿猪敌人 | 3 种体型，具有不同耐久度 |
| 结构物体 | 木块、石块、玻璃，耐久度和抗性各异 |
| 关卡系统 | 3 个关卡，每关 3 小节 |
| 计分系统 | 基于消灭猪猪数量和剩余小鸟数量计分 |

### 2.2 用户界面

- 主菜单（开始游戏、关卡选择、设置）
- 游戏界面（弹弓、发射轨迹、剩余小鸟显示）
- 暂停菜单
- 结算界面（星级评分、重玩/下一关）

### 2.3 音效与视觉效果

- 背景音乐和音效
- 粒子特效（爆炸、碰撞）
- 小鸟飞行轨迹显示

## 3. 技术架构

### 3.1 前端技术

- **渲染引擎**：HTML5 Canvas
- **物理引擎**：Matter.js
- **音频**：Howler.js

### 3.2 后端技术（如需）

- **服务器**：Node.js + Express
- **数据库**：SQLite（存储用户进度和分数）
- **API**：RESTful 接口

## 4. 验收标准

| 验收项 | 标准 |
|--------|------|
| 游戏可运行 | 在现代浏览器中无错误加载 |
| 物理效果 | 小鸟运动符合物理定律 |
| 碰撞检测 | 准确的碰撞判定和响应 |
| 关卡完整性 | 3 个关卡均可正常游玩 |
| 计分准确性 | 分数计算正确 |
| 界面美观 | 符合愤怒的小鸟视觉风格 |
