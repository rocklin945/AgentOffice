# 个人网站 PRD - 产品需求文档

## 1. 项目概述

### 1.1 项目名称
个人网站 (Personal Website)

### 1.2 项目目标
为个人/自由职业者打造一个专业的展示型网站，用于展示个人简介、作品集、技能、博客文章和联系方式。

### 1.3 目标用户
- 自由职业者
- 求职者/求职者
- 创业者/个人品牌
- 技术博主/内容创作者

---

## 2. 功能需求

### 2.1 首页 (Home)
- **Hero 区域**：大标题 + 副标题 + CTA 按钮 + 个人头像
- **简介区域**：简短的个人介绍文字（100-200字）
- **技能展示**：图标 + 技能名称列表
- **数据统计**：项目数、经验年限、客户数等

### 2.2 关于我 (About)
- 个人详细简介
- 教育背景
- 工作经历时间线
- 个人价值观/理念

### 2.3 作品集 (Portfolio)
- 作品卡片展示（图片 + 标题 + 描述）
- 分类筛选功能（全部/网站/移动端/设计）
- 点击查看作品详情弹窗
- 外链跳转功能

### 2.4 博客 (Blog)
- 文章列表页（标题 + 发布时间 + 摘要 + 标签）
- 文章详情页
- 分类筛选
- 分页功能

### 2.5 联系方式 (Contact)
- 联系表单（姓名、邮箱、主题、留言内容）
- 社交媒体链接（GitHub、LinkedIn、Twitter等）
- 地图占位区域

### 2.6 全局功能
- **导航栏**：Logo + 菜单项 + 暗色/亮色切换
- **页脚**：版权信息 + 快速链接 + 社交链接
- **响应式设计**：支持桌面、平板、手机

---

## 3. 非功能需求

### 3.1 性能要求
- 首屏加载时间 < 3秒
- 图片懒加载优化
- CDN 静态资源加速

### 3.2 兼容性
- Chrome / Firefox / Safari / Edge 最新两个版本
- iOS / Android 移动端适配

### 3.3 可维护性
- 代码模块化
- 清晰的目录结构
- 完善的注释

### 3.4 UI/UX 设计风格
- 现代简约风格
- 主色调可配置
- 柔和的动画过渡效果
- 卡片式布局

---

## 4. 技术选型

### 4.1 前端技术
- **HTML5 + CSS3 + JavaScript**
- 可选：Vue.js / React 框架

### 4.2 样式方案
- CSS Variables 管理主题
- Flexbox / Grid 布局
- CSS Animations

### 4.3 图标与字体
- Google Fonts（思源黑体/Inter）
- Font Awesome 或 Lucide Icons

---

## 5. 页面结构

```
/
├── index.html          # 首页
├── about.html          # 关于我
├── portfolio.html      # 作品集
├── blog.html           # 博客列表
├── contact.html       # 联系方式
├── css/
│   └── style.css      # 全局样式
├── js/
│   └── main.js        # 交互脚本
└── assets/
    ├── images/         # 图片资源
    └── icons/         # 图标资源
```
