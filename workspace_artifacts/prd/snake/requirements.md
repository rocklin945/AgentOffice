# 贪吃蛇游戏 - 产品需求文档 (PRD)

## 1. 项目概述

### 项目名称
贪吃蛇小游戏 (Snake Game)

### 项目类型
HTML5 休闲游戏

### 核心功能简介
一款经典复刻的贪吃蛇网页游戏，玩家通过方向键控制蛇的移动方向，让蛇吃食物增长，避开墙壁和自身碰撞。

### 目标用户
- 休闲游戏爱好者
- 怀念经典游戏的玩家
- 寻求快速娱乐的用户

---

## 2. 功能需求

### 2.1 游戏核心功能

#### 2.1.1 游戏区域
- 画布尺寸：400px × 400px
- 网格系统：20×20 像素的单元格
- 背景色：深色 (#1a1a2e)

#### 2.1.2 蛇的机制
- 初始长度：3节
- 初始位置：画布中央偏左
- 初始移动方向：向右
- 移动速度：150ms 每帧
- 身体颜色：渐变绿色 (#4ade80 → #22c55e)
- 蛇头颜色：深绿色 (#166534)

#### 2.1.3 食物机制
- 外观：红色圆形 (#ef4444)
- 大小：20×20 像素
- 生成规则：在空白格子随机生成
- 吃到食物后蛇身 +1 节，分数 +10

#### 2.1.4 碰撞检测
- 撞墙：游戏结束
- 撞自身：游戏结束
- 吃食物：蛇身增长，分数增加

#### 2.1.5 计分系统
- 初始分数：0
- 吃一个食物：+10 分
- 显示当前分数
- 显示最高分数（localStorage 存储）

### 2.2 用户交互

#### 2.2.1 控制方式
- 键盘方向键：↑ ↓ ← → 控制方向
- 空格键：开始游戏 / 重新开始
- ESC键：暂停游戏

#### 2.2.2 游戏状态
- **开始界面**：显示游戏标题和"按空格开始"
- **游戏中**：正常游戏进行
- **暂停**：显示"PAUSED"
- **结束**：显示"Game Over"和最终分数

### 2.3 界面设计

#### 2.3.1 视觉风格
- 整体风格：简约复古像素风
- 主色调：深色背景 + 明亮游戏元素
- 圆角：按钮和卡片使用 8px 圆角

## 3. 技术要求

### 3.1 技术栈
- **前端**：原生 HTML5 + CSS3 + JavaScript
- **画布**：Canvas 2D API
- **存储**：localStorage（最高分）

### 3.2 性能要求
- 流畅运行于现代浏览器
- 帧率稳定，无明显卡顿
- 响应式设计，适配不同屏幕

### 3.3 兼容性
- Chrome、Firefox、Safari、Edge 最新版本
- 桌面端优先（移动端可选）
