# 简单购物网站 PRD

## 1. 项目概述

- **项目名称**：SimpleShop - 简单购物网站
- **项目类型**：Web 电商应用
- **核心功能**：提供商品展示、购物车管理、商品搜索与分类浏览的轻量级购物平台
- **目标用户**：普通消费者，追求简洁购物体验的用户

---

## 2. 功能需求

### 2.1 商品展示

- 首页展示热门商品和推荐商品
- 商品列表支持网格/列表视图切换
- 商品卡片显示：商品图片、名称、价格、原价（划线）、折扣标签
- 商品详情页：展示商品大图、详细描述、规格参数、用户评价

### 2.2 商品分类

- 左侧导航栏展示商品分类菜单
- 支持多级分类（一级分类、二级分类）
- 分类页面展示该分类下的所有商品

### 2.3 搜索功能

- 顶部搜索框，支持关键词搜索商品
- 搜索结果页展示匹配的商品列表
- 支持按价格排序（升序/降序）

### 2.4 购物车

- 购物车图标显示商品数量角标
- 购物车页面展示已添加商品列表
- 支持修改商品数量（增加/减少）
- 支持删除商品
- 实时计算商品总价
- 显示已选商品总数量

### 2.5 用户模块

- 用户登录/注册页面
- 用户信息展示（头像、用户名）
- 模拟结账功能（生成订单）

---

## 3. 非功能需求

### 3.1 性能要求

- 页面首屏加载时间 < 3秒
- 操作响应时间 < 500ms

### 3.2 兼容性

- 响应式设计，适配 PC 端和移动端
- 主流浏览器兼容（Chrome、Firefox、Safari、Edge）

### 3.3 UI/UX 要求

- 界面简洁清晰，导航结构明确
- 色彩风格：电商主流风格，以白色为主色调，搭配蓝色/橙色点缀
- 交互流畅，有适当的动效反馈

---

## 4. 技术选型

### 4.1 前端

- **HTML5 + CSS3 + JavaScript**（原生实现）
- 使用 Flexbox 和 Grid 布局
- 无需框架，保持轻量

### 4.2 后端（模拟）

- 使用 localStorage 模拟本地数据存储
- 商品数据使用 JSON 文件存储
- 无需真实服务器

### 4.3 文件结构

```
shopping-site/
├── index.html          # 首页
├── css/
│   └── style.css       # 样式文件
├── js/
│   ├── app.js          # 主逻辑
│   ├── data.js         # 商品数据
│   └── cart.js         # 购物车逻辑
└── assets/
    └── images/         # 图片资源
```
