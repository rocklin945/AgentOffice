# 俄罗斯方块 (Tetris) 游戏需求文档

## 1. 项目概述

**项目名称**: Tetris  
**项目类型**: 经典街机游戏 (HTML5 单页应用)  
**核心功能**: 经典的俄罗斯方块消行游戏，包含方块旋转、下落、消行计分等核心玩法  
**目标用户**: 休闲游戏玩家，怀旧游戏爱好者

---

## 2. 游戏规则

### 2.1 方块类型
游戏包含 7 种经典俄罗斯方块 (Tetrominoes)：
- **I 型**: 4×1 横条
- **O 型**: 2×2 方块
- **T 型**: T字形
- **S 型**: S字形
- **Z 型**: Z字形
- **J 型**: J字形
- **L 型**: L字形

### 2.2 操作控制
| 操作 | 按键 |
|------|------|
| 左移 | ← 或 A |
| 右移 | → 或 D |
| 加速下落 | ↓ 或 S |
| 旋转 | ↑ 或 W |
| 硬降 (直接落到底) | 空格 |
| 暂停/继续 | P 或 Esc |

### 2.3 计分规则
- 消1行: 100 分
- 消2行: 300 分
- 消3行: 500 分
- 消4行 (Quad): 800 分
- 每消10行提升一级，速度加快

## 3. 界面设计

### 3.1 布局结构
```
┌─────────────────────────────────────┐
│            TETRIS                   │
├───────────────────┬─────────────────┤
│                   │   NEXT          │
│                   │  ┌─────┐        │
│    游戏区域        │  │     │        │
│    10×20          │  └─────┘        │
│                   ├─────────────────┤
│                   │   SCORE          │
│                   │   00000         │
│                   ├─────────────────┤
│                   │   LEVEL         │
│                   │   01            │
│                   ├─────────────────┤
│                   │   LINES         │
│                   │   000           │
└───────────────────┴─────────────────┘
```

### 3.2 视觉风格
- **主题**: 复古霓虹风格
- **背景色**: 深紫蓝色渐变 (#1a0a2e → #0d0015)
- **方块颜色**: 每个形状对应不同亮色（青、黄、紫、橙、蓝、绿、红）
- **边框**: 霓虹发光效果
- **字体**: 等宽像素风格

---

## 4. 技术规格

### 4.1 技术栈
- **前端**: 纯 HTML5 + CSS3 + Vanilla JavaScript
- **渲染**: Canvas 2D
- **无需后端**: 单文件即可运行

### 4.2 游戏参数
- 游戏区域: 10 列 × 20 行
- 每格大小: 30px × 30px
- 游戏区域尺寸: 300px × 600px
- 初始下落速度: 1000ms (1秒)
- 速度提升: 每级减少 10% 下落时间

### 4.3 音效
- 背景音乐: 无 (可静音)
- 音效: 消行、升级、游戏结束

---

## 5. 游戏状态

| 状态 | 说明 |
|------|------|
| START | 开始界面，按空格开始 |
| PLAYING | 游戏进行中 |
| PAUSED | 游戏暂停 |
| GAME_OVER | 游戏结束，显示最终分数 |
